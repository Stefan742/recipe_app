# Recipe App (Flutter) — Лабораториска вежба 2

Ова е целосен Flutter проект кој ја имплементира лабораториската вежба за прикажување рецепти користејќи TheMealDB API.

## Функции
- Почетен екран: Листа од категории (картички)
- Екран со јадења по категорија (grid)
- Детален приказ на рецепт (image, instructions, ingredients, YouTube button)
- Random рецепт преку AppBar икона
- Search за категории и за јадења
- Организирано во модели, services, screens, widgets, theme
- Cached images, basic error/loading handling

API: https://www.themealdb.com/api.php
